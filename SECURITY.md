# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (`main`) | ✅ |
| older tags | ❌ |

## Reporting a Vulnerability

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities via [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)
or email **dogukanmetan@gmail.com** directly.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or proof-of-concept
- Affected workflow files and versions

You will receive a response within **48 hours** confirming receipt, and a fix
or mitigation plan within **7 days** for confirmed vulnerabilities.
